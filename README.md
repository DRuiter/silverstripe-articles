# SilverStripe Articles

## Introduction
Article page type, can be used as news system or as a blog

## Requirements
* silverstripe/cms ^4.0
* silverstripe/lumberjack ^2.0

## Installation
```
composer require thewebmen/silverstripe-articles
```
